Alunos: Luan Carlos e Carlos Wesley 5. Então, o modelo monolítico é quando tudo fica junto no mesmo sistema, tipo frontend, backend e banco de dados tudo no mesmo lugar. Já o modelo frontend + backend via API separa as coisas: o frontend cuida da parte visual e o backend manda os dados por API. Exemplo: monolítico pode ser PHP com Laravel, e separado pode ser React com Node.js. 6. O DOM é tipo uma representação do HTML que o navegador cria em forma de árvore. Ou seja, a gente escreve o HTML, mas o navegador transforma isso em objetos que o JavaScript consegue mexer, tipo alterar texto, cor, etc. 7. O Box Model é como se fosse uma caixa que envolve os elementos. Tem o content (conteúdo), padding (espaço interno), border (borda) e margin (espaço externo). A diferença é que o padding aumenta o espaço dentro do elemento e o margin aumenta o espaço fora dele. 8. Bootstrap é um framework que já vem com vários estilos prontos pra facilitar a vida. A gente usa porque economiza tempo e já ajuda a deixar o site responsivo. Exemplo: .container pra organizar layout e .btn pra estilizar botões. 9. Os seletores podem ser por elemento (tipo p), por classe (.classe) e por id (#id). O id tem mais prioridade, depois a classe e por último o elemento. 10. HTML semântico usa tags com significado tipo , , e , enquanto usar só div não diz nada. Isso ajuda no SEO e também na acessibilidade, porque facilita pro navegador e leitores de tela entenderem melhor a página.
